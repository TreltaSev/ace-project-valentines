# ACE Project Assignment
## Valentines Site

### Setup Development:

1. Install Siblink
```
pip install git+https://github.com/TreltaSev/siblink.git
```

2. Run Initialization
```
siblink run init
```

### Commands:

- Run Development:
```
siblink run --ps dev
```

- Push Release:
```
siblink run scripts/release -- "Release Message"
```
